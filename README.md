# Enterprise SOC Home Lab

## Overview
This project simulates a small enterprise cybersecurity environment using Active Directory, Windows 11, Kali Linux, Splunk SIEM, and segmented networks.

The goal of this lab is to gain hands-on experience with:
- Active Directory administration
- Windows domain environments
- Network segmentation
- SIEM monitoring
- Security event analysis
- Incident response
- Vulnerability testing

---

# Lab Environment

## Infrastructure
- Windows Server 2022
- Windows 11 Pro
- Kali Linux
- Splunk Enterprise
- VMware Workstation
- ASUS Router
- Segmented Wireless Networks
